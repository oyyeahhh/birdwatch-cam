# Curriculum

Teacher-facing lesson plans and professional-development decks for the
BirdWatch Cam project, in the CIJE Birdwatcher design system.

These are classroom/PD deliverables — distinct from the technical design
docs in [`../docs/`](../docs/) and the vendor DFRobot material in
[`../ai_camera/docs/`](../ai_camera/docs/).

## Contents

| Deck | Audience | Topic |
|---|---|---|
| `CIJE_Birdwatcher_Onboarding.pptx` | teachers / students | Getting started with the BirdWatch Cam project |
| `CIJE_Birdwatcher_TeacherPD.pptx` | teachers (PD) | Professional-development walkthrough |
| `who_sees_the_birds_lesson.pptx` | students | Data sharing / privacy — the camera's Local / Diagnostics / Full Cloud modes |
| `github_open_source.pptx` | students | GitHub and open source |
